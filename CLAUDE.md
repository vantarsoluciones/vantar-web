# Vantar Soluciones - Página web principal

## Contexto del negocio
Agencia de marketing especializada exclusivamente en estudios jurídicos y abogados.
Servicios: Google Ads + Meta Ads + Agente de IA que entrevista leads y agenda turnos automáticamente.
Propuesta de valor: más clientes, menos tiempo perdido en consultas no calificadas.
Precio único: suscripción de USD 1.000/mes.

## Identidad visual (extraída del logo)
- Azul oscuro principal: #1B3A8F
- Cian/turquesa accent: #00D4D4
- Gris oscuro texto: #4A4A4A
- Fondo: oscuro preferentemente (transmite tecnología)
- El logo tiene circuitos impresos en la V → usar ese motivo tecnológico en el diseño

## Skills
- Siempre invocar /frontend-design antes de construir cualquier componente visual
- Comprometerse con una aesthetic direction antes de escribir código

## MCPs disponibles
- Usar Context7 para consultar documentación de Next.js y Tailwind
- Usar Playwright para tomar screenshots de localhost:3000 y verificar resultado visual

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Deploy: Vercel (más adelante)

## Reglas de diseño
- Estética: tecnológica, profesional, confiable. NO startup genérica.
- NUNCA usar Inter o Roboto como tipografía display
- Animaciones sutiles que refuercen la sensación de tecnología (partículas, líneas de circuito, gradientes animados)
- Paleta dominante azul oscuro + accent cian
- Layout limpio con mucho espacio negativo
- Botón/CTA principal: ícono de WhatsApp que abre chat

## Tono del copy
- Español argentino, voseo
- Directo y orientado a resultados ("más clientes", "agenda llena", "sin perder tiempo")
- Profesional pero no frío
- Dirigido a abogados: entienden de negocios, valoran resultados concretos

## Secciones de la página (en orden)
1. Hero: propuesta de valor principal + CTA WhatsApp
2. Servicios: Google Ads, Meta Ads, Agente IA de calificación y agendamiento
3. Cómo funciona: flujo paso a paso (lead entra → IA lo entrevista → turno agendado)
4. Resultados/Casos de éxito: placeholder temporal con métricas genéricas creíbles
5. Precio: suscripción única USD 1.000/mes, qué incluye
6. Footer con WhatsApp CTA
